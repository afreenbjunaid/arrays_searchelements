# arrays_searchelements
This is a Java program that checks whether two arrays contain the same unique elements.

Example Input Arrays:

- `Integer[] array1 = {1, 2, 3, 2, 1};`
- `Integer[] array2 = {1, 2, 3};`

Expected Output:

The program should return `true` for the above inputs.

## Java Program

A sample Java program is provided in `arraySearchCmpUniqElems.java`. It checks whether two arrays contain the same unique elements regardless of order and duplicates.

### Compile and Run

```bash
javac arraySearchCmpUniqElems.java
java arraySearchCmpUniqElems
```
